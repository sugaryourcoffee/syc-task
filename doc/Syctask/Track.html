    class Syctask::Track

    A Track holds a task and stops the time the task is processed. The Track will print every second the elapsed time and the time left to the specified Syctask::Task#duration.

    Attributes

    dir[R]

    Directory where the tracked task is saved

    id[R]

    ID of the tracked task

    started[R]

    When tracking started

    stopped[R]

    When tracking stopped

    title[R]

    Title of the tracked task

def initialize(task)
  @dir = task.dir
  @id = task.id
  @title = task.title
  @duration = task.remaining.to_i
  @semaphore = "#{Syctask::SYC_DIR}/#{@id}.track"
end

    Starts the tracking and a timer that will print to STDOUT every second the elapsed time and the time left until Syctask::Task#duration

def start(show)
  @started ||= Time.now
  # start a timer that prints id and elapsed time 
  FileUtils.touch @semaphore
  system "console_timer #{@duration} #{@id} #{@semaphore} &" if show
end
    stop() click to toggle source

    Stops the task tracking and returns the lead time of the task

def stop
  FileUtils.rm @semaphore if @semaphore and File.exists? @semaphore
  @stopped ||= Time.now
  @stopped - @started
end
